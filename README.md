# NX Time Sync
Simple app to sync the system clock via the internet.

## Credits
- [@thedax](https://github.com/thedax) for NX-ntpc, from which the parent project was forked.
- [@3096](https://github.com/3096) for switch-time, from which this project is forked.

Icon from [Openclipart](https://openclipart.org/detail/198121/mono-ntpclient)

## Building
Install [devkitA64 (along with libnx)](https://devkitpro.org/wiki/Getting_Started), and run `make` under the project directory.
